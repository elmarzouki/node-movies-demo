# node-movies-demo
Movies app demo using node and mongoDB
Expected that you have a running mongo server.

Run
-----
  1. `$ npm run start`
  2. GET localhost:3000/movies
  3. POST localhost:3000/movies
  `json
    {
      {
          "__v": 0,
          "name": "Movie name",
          "genre": [
              "comedy"
          ]
      }
    }
  `
